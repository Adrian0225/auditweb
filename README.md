# auditweb


## Funkcje
- Weryfikacja HTTPS
- Sprawdzanie nagłówków bezpieczeństwa
- Analiza formularzy <form>
- Analiza cookies 
- Wyszukiwanie potencjalnych problemów w HTML
- Generowanie raportu JSON

## Instalacja
pip install -r requirements.txt

## Obsługa
Start: python auditproject.py

Program wyświetli okno "Podaj URL" i okno na input.

Po wpisaniu przeanalizuje ten adres i stworzy plik json z raportem
